# 1단계 - 지뢰 찾기(그리기)
### 기능 요구사항

* 높이와 너비, 지뢰 개수를 입력받을 수 있다.
* 지뢰는 눈에 잘 띄는 것으로 표기한다.
* 지뢰는 가급적 랜덤에 가깝게 배치한다.

### 기능 목록

- [x] 입출력 담당 클래스 
- [x] 빈칸, 지뢰를 담당하는 셀[Cell]
- [x] 셀들의 조합인 지뢰 게임 맵[Map] 


# 2단계 - 지뢰 찾기(지뢰 개수)
### 기능 요구사항
> 1단계 기능 요구사항에서 추가 되다.
* 각 사각형에 표시될 숫자는 자신을 제외한 주변 8개 사각형에 포함된 지뢰의 개수다.

### 기능 목록

- [x] Blank Cell 에 지뢰 갯수를 기록 하도록 수정하다.
